# Bermon Group – Sitio Corporativo

Sitio web corporativo de Bermon Group, orientado a servicios de
seguridad, análisis de riesgos y tecnología aplicada.

El sitio está diseñado como una aplicación web estática, liviana,
segura y fácilmente desplegable en cualquier servicio de hosting.

---

## Estructura del proyecto

bermon-group/
- index.html
- css/
  - styles.css
- js/
  - main.js
- img/

---

## Tecnologías utilizadas

- HTML5
- CSS3
  - Diseño responsive
  - Variables CSS
  - Modo claro / oscuro
- JavaScript (Vanilla)
- Sin frameworks ni dependencias externas

---

## Objetivo del sitio

- Presentación corporativa de Bermon Group
- Comunicación clara de servicios y metodología
- Base técnica sólida para futuras ampliaciones
  (paneles, formularios, aplicaciones, etc.)

---

## Estado del proyecto

- Sitio funcional
- Responsive (desktop / mobile)
- Modo oscuro implementado
- Listo para despliegue en hosting estático

---

## Consideraciones técnicas

- No requiere backend
- No almacena datos
- No utiliza cookies
- Superficie de ataque mínima
- Apto para entornos corporativos
